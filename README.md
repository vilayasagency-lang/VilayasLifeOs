# LifeOS — Billion-Dollar Personal Infrastructure

## 🚀 Setup Guide
1. **Supabase:** Create project, run SQL Master Script, Disable Email Confirmation.
2. **Cloudflare:** Create R2 bucket `lifeos-assets`, deploy Worker, Bind R2.
3. **Frontend:** Update `js/supabase-config.js` and `js/api.js` with your live URLs.
4. **Deploy:** Push to GitHub and connect to Vercel/Cloudflare Pages.

## 📱 Features
- Encrypted Vault (R2 Powered)
- Family SOS Hub
- Smart Expense Tracking
- Multi-language Support
- installable PWA App
